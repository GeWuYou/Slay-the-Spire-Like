一个使用 **Godot 4** 与 **C#** 制作的卡组构筑、回合制战斗原型项目，灵感来源于 *Slay the Spire*，并基于中级卡牌战斗教程构建。

本项目为原版 GDScript 教程代码的 **C# 重写版本**。

## 🎯 项目目标

- 使用 **C# 完整重构** 教程中的卡牌战斗系统  
- 改进整体架构（SOLID、组件化、低耦合）  
- 搭建可复用的卡牌构筑/回合制游戏框架  
- 为 Godot + C# 开发者提供参考示例

| 组件         | 技术说明                       |

| 编程语言     | **C# (.NET 8 via Godot Mono)** |
| 核心架构模式 | OOP + 基于组合的实体系统       |
| UI           | Godot Control 节点             |
| 输入         | Godot InputMap                 |
| 构建         | .NET SDK + Godot Export        |

---

## 💡 为什么选择 C# 重写？

Godot 的 C# API 提供了：

- 更强的静态类型系统  
- 更适合大项目的代码管理  
- .NET 生态兼容性  
- 在大量逻辑处理中更具性能优势  
- Rider / VS / VS Code 具备更强的重构能力
